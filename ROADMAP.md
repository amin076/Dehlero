# Dehlero Roadmap

## Phase 1 — Foundation

- Vite + TypeScript + Three.js
- Theatre.js integration
- Camera controls
- Basic scene loading
- Basic object loading
- Manual video recording

## Phase 2 — Studio Controls

- Lighting controls
- Camera presets
- Scene selector
- Object visibility controls
- Long video and Shorts presets
- Safe area overlays

## Phase 3 — Timeline and Director

- Shot definitions
- Timeline playback
- Automatic camera movement
- Programmed video generation
- Auto record start/stop

## Phase 4 — Scientific Asset Library

- Astronomy objects
- Optics objects
- Earth science objects
- Physics objects
- Spacecraft and satellites

## Phase 5 — Advanced Production

- Scenario generator
- Reusable video templates
- Voiceover support
- Captions and overlays
- Batch video generation
