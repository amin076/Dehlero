# Dehlero Studio Design

## Version 1 Layout

Dehlero v1 uses a fullscreen 3D viewport with floating panels.

## Main Layout

- Center: 3D viewport
- Top-left: scene/object toolbar
- Top-right: camera and lighting controls
- Bottom: timeline / Theatre.js panel
- Bottom-right: recording controls

## Studio Modes

### Manual Mode

The user manually controls camera, objects, lighting, and recording.

### Director Mode

The system plays a programmed scenario with camera movement, lighting, timing, and automatic recording.

## Core Panels

- Asset Browser
- Object List
- Camera Panel
- Lighting Panel
- Timeline Panel
- Recording Panel
- Safe Area Overlay Panel

## Video Presets

- Shorts 9:16
- Landscape 16:9
- Square 1:1
- Custom
