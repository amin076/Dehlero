# Saturn Asset

This folder contains the Dehlero Saturn asset.

Expected files:

- textures/saturn_albedo.jpg
- asset.json

Saturn is procedural:
- sphere geometry is created by Dehlero
- texture is applied to the sphere
- rings use the existing Dehlero Saturn rings generator

Used for:
- astronomy scenes
- Titan World Cup 2226 trailer
- Saturn / Titan cinematic shots
